HTML and CSS Cheatsheet

Key HTML tags

HTML Tags Name Brief description
<!DOCTYPE html> Document Type This html tag declares the document type and version of HTML, ensuring the browser renders the page in standards mode.
<html></html> html Root element that encompasses the entire HTML document. A closing tag is required.
<head></head> head Contains meta-information and links to resources. A closing tag is required.
<body></body> body Contains the content of the document. A closing tag is required.
<link> link This html tag defines a relationship between the current document and an external resource, typically used to link stylesheets. A closing tag is not required.
<footer></footer> footer This html tag defines a footer for a document or section, typically containing author information, copyright details, or navigation links.

Key CSS Properties

CSS Property Name Brief description
color color Sets the text color
background-color background color Sets the background color of element/page
font-size font size Sets the size of the text
width width Sets the width of the element
height height Sets the height of the element
